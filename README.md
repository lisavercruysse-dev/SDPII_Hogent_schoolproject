# Planning tool application

A web and desktop application to manage employees, their absences and their tasks. Also production sites and their machines can be managed. The project is part of school assignment where we built a full application (web and desktop) for a real world company.

The main challenges with this project was trying to navigate the sometimes uncertain needs of the client, pinpoint what is expected exactly and working together in a team to work as efficiently as possible. I took a more leading role upon myself to try deliver an end result of a certain quality. 

_Note: There are definitely still some bugs but there wasn't enough time to fix everything as we were a group with 1 person less._
